Factory Mania!

By: Jakub Zahora

Application Overview

User starts the game by creating an account. After that, they are greeted by a welcome screen, and the game officially begins.

User has to mine iron, which they will then smelt into iron bars. After acquiring enough iron bars, they will be able to create a

mining drill, which will mine ore even when user is not playing the game. When user comes back to the game, appropriate amount of items

will be added to the user account, at which point they will be able to spend it how they wish. Game continues through unlocking new types of ore,

which allow creation of more complex items, with the game ending when the user will amass enough items to create a rocker ship.
